# Homebrew tap — Daimoni

Homebrew formulas for the Daimoni stack.

## Install

```bash
brew install marlic1982/tap/daimoni-agent
```

## Formulas

- `daimoni-agent` — local daemon for folder sync (modalità Progetti)
  + bridge mode (modalità Processi). See
  [docs/manuali](https://github.com/marlic1982/Daimoni/tree/main/docs/manuali).
